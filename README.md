# Antigravity - 레이아웃 파싱 개발 프로젝트

이 프로젝트는 현대적이고 세련된 디자인의 랜딩 페이지를 구축하고, 동적인 인터랙션과 반응형 레이아웃을 구현하는 프로젝트입니다.

## 🚀 주요 기능

### 1. 동적 테마 인터랙션
- 중앙의 메인 카드에서 버튼을 클릭할 때마다 문구와 테마 색상이 동기화되어 변경됩니다.
- 부드러운 애니메이션 효과와 함께 텍스트 색상과 버튼 배경색이 함께 변화합니다.

### 2. 세련된 디자인 시스템
- **컬러 스킴**: 눈의 피로를 최소화하는 부드러운 녹색(Soft Green) 배경과 신뢰감을 주는 어두운 슬레이트(Dark Slate) 색상의 조화.
- **타이포그래피**: `Noto Sans KR` 폰트를 적용하여 한글 가독성을 최적화하고 명확한 시각적 위계를 구성했습니다.
- **애니메이션**: 마우스 호버 시 버튼의 크기 확대 및 색상 심화 효과를 통해 생동감 있는 사용자 경험을 제공합니다.

### 3. 완벽한 반응형 레이아웃
- 데스크톱 뿐만 아니라 모바일 환경에서도 최적화된 화면을 제공합니다.
- 768px 이하 화면에서 헤더와 본문 요소들이 자연스럽게 세로로 스태킹됩니다.
- 모바일 사용자를 위해 버튼 크기를 터치하기 좋은 크기로 최적화했습니다.

### 4. 사용자 후기 섹션
- 프로필 이미지와 함께 신뢰도 있는 사용자 후기를 카드 형태로 구현했습니다.
- 이미지 경로 최적화를 통해 로딩 성능과 안정성을 확보했습니다.

## 🛠 기술 스택
- **HTML5**: 시맨틱 태그를 활용한 웹 표준 구조
- **CSS3**: Vanilla CSS를 활용한 디자인 시스템 및 미디어 쿼리(반응형)
- **JavaScript**: 실시간 텍스트 및 클래스 조작을 통한 동적 인터랙션
